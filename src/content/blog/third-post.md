## A Bilingual Blog Post

This is the English version of the third blog post.
It demonstrates how the bilingual content system works.
You can switch between English and Korean using
the language toggle in the header.

### How It Works

Each blog post can contain content in both languages,
separated by special `<!-- lang:en -->` and `<!-- lang:ko -->` markers.
The toggle button in the header switches between them instantly.

## 이중 언어 블로그 포스트

이것은 세 번째 블로그 포스트의 한글 버전입니다.
이중 언어 콘텐츠 시스템이 어떻게 작동하는지 보여줍니다.
헤더의 언어 토글을 사용하여 영어와 한글을 전환할 수 있습니다.

### 작동 방식

각 블로그 포스트는 두 가지 언어로 콘텐츠를 담을 수 있으며,
특별한 `<!-- lang:en -->` 과 `<!-- lang:ko -->` 마커로 구분됩니다.
헤더의 토글 버튼으로 즉시 전환됩니다.
